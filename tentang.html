Tentang Saya

Halo! Nama saya Gracia Inri, saya adalah mahasiswa Sistem Informasi yang memiliki ketertarikan besar dalam dunia teknologi dan inovasi digital. Saya memiliki semangat belajar yang tinggi dan selalu ingin mengembangkan kemampuan saya khususnya dalam bidang teknologi informasi.


Saya sangat tertarik pada dunia coding dan pengembangan website. Saya senang mempelajari hal-hal baru seperti HTML, CSS, JavaScript, database, dan pembuatan sistem berbasis web. Bagi saya, coding bukan hanya tugas kuliah tetapi juga sebuah kegiatan yang menyenangkan karena saya bisa menciptakan sesuatu dari nol menjadi sebuah aplikasi yang bisa digunakan oleh orang lain. Selain itu, saya juga merupakan pribadi yang suka mencoba hal baru. Saya percaya bahwa pengalaman baru dapat membantu saya berkembang menjadi pribadi yang lebih baik. Saya tidak takut mencoba sesuatu yang belum pernah saya lakukan sebelumnya karena dari situlah saya bisa belajar banyak hal.

Saya sering mengerjakan berbagai proyek coding seperti pembuatan website, aplikasi berbasis database, serta aplikasi sederhana menggunakan JavaScript. Dari kegiatan coding tersebut saya belajar tentang problem solving, logika berpikir, dan bagaimana membuat sistem yang bermanfaat.

Selain aktif di bidang teknologi, saya juga merupakan seorang content creator affiliate. Saya membuat konten di media sosial untuk membagikan pengalaman menggunakan produk serta memberikan rekomendasi kepada orang lain. Sebagai affiliate, saya belajar banyak hal seperti komunikasi digital, kreativitas, konsistensi, serta bagaimana memahami kebutuhan audiens. Pengalaman ini membantu saya menjadi pribadi yang lebih kreatif dan percaya diri. Saya percaya bahwa teknologi dan kreativitas bisa berjalan bersama. Oleh karena itu saya mencoba mengembangkan diri baik di bidang teknologi maupun di bidang digital content. Ke depannya saya ingin terus berkembang menjadi individu yang mampu memberikan dampak positif melalui teknologi. Saya ingin belajar lebih banyak, berkolaborasi dengan banyak orang, serta ikut berkontribusi dalam perkembangan teknologi digital.
